# こみゃくのおと (Co-MYAKU Note)

**Version 2.2.0**

「こみゃくのおと」は、2025年日本国際博覧会（大阪・関西万博）の公式キャラクターからインスパイアされた、非公式のインタラクティブ・サウンド・ファンアート（二次創作）です。

Web Audio APIによる自動生成アンビエント・ミュージックと、Canvas APIによる有機的なビジュアル・インタラクションを融合させた、ブラウザで体験するデジタル・トイです。

---

## ■ 主な機能

### 1. ジェネレーティブ・オーディオ

ピアノ、ストリングス、ベースの3パートが、それぞれ異なる素数周期（7/11/13/17/19）のアルゴリズムに従って音を奏でます。v2.2.0では各楽器音色を完全に再設計しました。

**楽器仕様**
- **Piano：** エレクトリックピアノ風の倍音設計。暖かくこもった音を実現します。
- **Strings：** ストリングアンサンブル風ビンテージ・シンセストリングス。三相BBDコーラスをシミュレート。
- **Bass：** ベースシンセサイザー風。SAW波のデチューンとLPFスイープで、アタック後の沈み込みが特徴。

**視線連動・コンタクトサウンド**
- 画面中央のこみゃくの視線の高さに応じて、音の密度（アクティビティ）が動的に変化します。
- 画面をタップするとEメジャースケールからランダムに音が選ばれ、ユーザー自身が演奏に参加できます。

### 2. ダイナミック・ビジュアル

- **視線追従＋脈動:** タッチやマウス操作に合わせて視線が動き、瞳の大きさがリアルタイムの音量レベルで脈動します。
- **パーティクル生成:** 各楽器の発音タイミングに1:1で連動して放射状パーティクルが生成。赤（ピアノ）・青（ストリングス）・灰色（ベース）の色分け。

### 3. オーディオシステム

**リバーブタイプ（切り替え可能）**
- **Metro（地下鉄駅コンコース）:** 巨大地下空間をシミュレート。RT60≈4.5s。
- **Pavilion（膜構造空間ステージ）:** 膜素材が高周波を吸収し、低周波・膜固有振動が長く残る暗く柔らかい残響。RT60≈2.8s。

### 4. カスタマイズ機能（設定メニュー）

右上の設定アイコンから、体験を自分好みに調整できます。

**ボリューム・テンポ・チャンネル**
- **Master Volume:** -60dB～0dBの全体音量調整。
- **Tempo:** 1/1（120BPM）と1/2（60BPM）の切り替え。楽曲の表情をガラリと変更。
- **Mute Channels:** ピアノ・ストリングス・ベースのパート別ミュート。ソロ演奏のような構成も楽しめます。

**背景・リバーブ**
- **Background Style:** 最小限の「WHITE」背景、または「PHOTO SELECT」からお気に入りの写真を背景に設定（最大1600px自動ダウンスケール）。
- **Reverb Type:** MetroとPavilionの切り替え。

## ■ 使い方

1. ブラウザで `index.html` を開きます。
2. タイトル画面の「START」をクリックして開始します。
3. 画面をタップ、またはドラッグして、音と視覚の反応を楽しんでください。
4. 右上の設定メニューから、ボリューム・テンポ・背景・リバーブタイプをカスタマイズできます。
